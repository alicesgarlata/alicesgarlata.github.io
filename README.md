# Alice Sgarlata — Portfolio website

A responsive, single-page portfolio for a multidisciplinary creative working across graphic design, illustration and visual identity.

## Preview

Open `index.html` in a browser. For the most reliable local preview, run a simple local server from this folder, for example with VS Code Live Server.

## Before publishing

1. Replace `hello@example.com` in `index.html` with the correct email address.
2. Confirm the venue Instagram link in `script.js`.
3. Add the name, medium and year of the observational drawing when available.
4. Review the provisional English project descriptions.

The Digital Projects section links to the live university websites and Alice's GitHub repositories. Featured projects are Lima Live Cinema Museum, Benozzo Gozzoli Knowledge Graph and God of War LODLAM.

The site is suitable for deployment through GitHub Pages without a build step.
